    Section 3｜美股族群分組


    A. CPU / General Server：Intel、AMD

    本週觀察：一般 server CPU 沒有壞，但市場焦點明顯被 AI server 吸走。AMD 強於 Intel。

    下週看法：可延續，但多屬財報前預期交易，映射台股時只能給中等權重。

    產業關鍵句：一般 server 有復甦，但估值話語權已讓位給 AI server。

    風險點：企業 IT 支出回升若不如預期，General Server 鏈會先被市場降評價。


    B. AI GPU / ASIC / High-speed Networking：Nvidia、Broadcom、Marvell、Arista

    本週觀察：最強主線。AI 晶片、ASIC、自研加速器、高速網路與 rack-level 互連需求同步強。

    下週看法：仍是首選映射方向；但股價已熱，台股只能挑「可回踩 / 可再突破」者。

    產業關鍵句：AI 基礎設施已從買 GPU，升級成 GPU + ASIC + Networking + Optical + Rack integration 全套競賽。

    風險點：capex 回收期若被市場質疑，最先修正的也是高估值 AI infra。


    C. Memory / Storage：Micron

    本週觀察：AI memory / HBM 仍偏緊，記憶體敘事保持偏多。

    下週看法：可續強，但本輪台股候選池內沒有最純的 HBM 映射標的，因此列為次映射。

    產業關鍵句：AI 正把記憶體從景氣循環股，推向供給受限資產。

    風險點：若擴產速度過快、終端拉貨遞延，記憶體股波動會很大。


    D. Passive / Power / Industrial：Vishay、VPG、Eaton、Amphenol

    本週觀察：AI datacenter 的電力、散熱、連接器需求強；但傳統工業景氣仍偏普通。

    下週看法：台股映射重點在 Power / Power Shelf / PSU / 基礎設施，不在一般工控。

    產業關鍵句：AI server 吃的不只晶片，還吃更多電力、熱管理與連接件。

    風險點：若 datacenter 建置延後，電源與基礎設施鏈通常晚於晶片反映。


    E. AI Software / Cloud Capex：Microsoft、Meta、Amazon、Google、Cloudflare、Palantir

    本週觀察：Big Tech AI capex 仍大；台股 AI 供應鏈的最核心驅動，仍是 hyperscaler 開支。

    下週看法：這條線最值得映射到台股 ODM / 網通 / 散熱。

    產業關鍵句：AI 軟體/雲端的本質不是故事，而是 capex 是否繼續轉成硬體拉貨與營收兌現。

    風險點：若市場開始質疑 capex ROI，台股 AI 硬體鏈會同步受壓。


    F. Consumer / Terminal：Apple、Tesla

    本週觀察：Apple 偏穩，Tesla 高波動。

    下週看法：這條線對本週台股 watchlist 幫助不大，不作主映射。

    產業關鍵句：終端不是全面復甦，而是品牌與產品策略分化。

    風險點：終端需求能見度仍不如 AI infra 清楚。
