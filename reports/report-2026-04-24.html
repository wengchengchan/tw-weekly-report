Section 3｜美股族群分組


A. CPU / General Server

  • 代表股：Intel、AMD
  • 本週觀察：AMD 明顯強於 Intel，代表 CPU / General Server 並非沒有需求，而是市場給了更高權重給能連到 AI 的那一側。
  • 下週看法：中性偏多，但只能列次主線。
  • 產業關鍵句：General Server 有復甦，但資金話語權仍在 AI Server。
  • 風險點：Intel earnings 若偏弱，整個 CPU 組會被拖累。

B. AI GPU / ASIC / High-speed Networking

  • 代表股：Nvidia、Broadcom、Marvell、Arista
  • 本週觀察：SOX 週漲 +8.03%，其中 Marvell +24.14%，Broadcom +5.39%，Arista +7.17%；市場交易重心持續往 custom silicon、network fabric、rack-level integration 移動。
  • 下週看法：仍是最強主線，也是最值得映射到台股的第一順位。
  • 產業關鍵句：AI 基建已經從「買 GPU」變成「整個 rack stack 一起買」。
  • 風險點：估值與追價風險很高，強股一旦失速，台股高位股跌幅也會很大。

C. Memory / Storage

  • 代表股：Micron
  • 本週觀察：Micron 週漲 +5.36%，AI memory 緊張與高頻寬需求敘事仍延續。
  • 下週看法：可列中高延續主線，但台股本次核心池沒有最純的 HBM / memory 映射股，因此只列次映射。
  • 產業關鍵句：AI 正把記憶體從 cyclical 變成帶有稀缺性溢價的資產。
  • 風險點：一旦市場預期供給鬆動，記憶體股修正速度通常很快。

D. Passive / Power / Industrial

  • 代表股：Vishay、VPG、Eaton、Amphenol
  • 本週觀察：VPG +12.48%、ETN +8.09%，Power / Industrial 中與 datacenter 關聯度高的名字繼續強；一般工業仍偏平。
  • 下週看法：可列保留主線，但只映射 AI datacenter power / rack / connector，不映射一般工控。
  • 產業關鍵句：AI server 不只吃算力，也吃更多電、更多散熱、更多連接。
  • 風險點：若 hyperscaler 建置時程遞延，power / infrastructure 鏈通常會晚一步補跌。

E. AI Software / Cloud Capex

  • 代表股：Microsoft、Meta、Amazon、Google、Cloudflare、Palantir
  • 本週觀察：股價表現本週不算強，但這條線的真正重點不在股價，而在 4/30 earnings 對 capex 的指引。
  • 下週看法：延續性仍高，且仍必須映射到台股 ODM / 網通 / 散熱。
  • 產業關鍵句：Cloud capex 才是台股 AI 硬體鏈最上游的訂單來源。
  • 風險點：若 Big Tech 對 capex ROI 轉保守，高位台股 AI 股會同步去化估值。

F. Consumer / Terminal

  • 代表股：Apple、Tesla
  • 本週觀察：AAPL +3.81%、TSLA -3.90%，內部分化很大。
  • 下週看法：不列本週主映射。
  • 產業關鍵句：終端不是沒有機會，但現在不是市場最願意出高本益比的方向。
  • 風險點：與台股本週 AI watchlist 的因果鏈太弱。

Section 8｜M.E.T.A. 檢查清單與定義


M.E.T.A. = Multiple Edge Trading Area

字母意義核心說明
M（Multiple）多重優勢不做單一訊號交易，至少 3 個 Edge 重疊才成立
E（Edge）優勢來源MA、flip、趨勢線、量縮、法人、抗跌都屬 Edge
T（Trading）可交易必須能給出買點、停損、目標，而不是只寫研究心得
A（Area）區域MA 是帶狀區，不是精準到 0.5 元的神奇價位

本週實務結論

  • A 級不等於現價可追；A 只代表 等回到交易區時值得優先做。
  • 如果股價已高於 M.E.T.A. 區上緣 × 1.03，即使基本面再強，也只能降成「等拉回 / 等突破」。
  • 本週最常見的失敗原因不是主線錯，而是 離區太遠、風報比太差。
